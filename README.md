# arduinaNano-Distance_Measurement Using Ultrasonic sensor and 4-digit-display

Requirements:

    1.HC-SR04 Ultrasonic Sensor
    2.TM1637 4-digit Display
    3.Arduino Nano
    4.Some Jumper wires
    5.BreadBoard
 Connections:
 Trig pin - D8
 Echo pin - D9
 VCC - 5v (Both ultrasonic and Display)
 GND - GND
 
 Display:
 clk - D2
 DIO - D3
 GND - GND
 VCC - 5v
